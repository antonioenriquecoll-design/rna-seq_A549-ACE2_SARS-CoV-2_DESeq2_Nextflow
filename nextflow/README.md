# Implementación en Nextflow

Esta carpeta contiene los archivos principales de la pipeline complementaria en Nextflow:

- `main.nf`: código principal del workflow en Nextflow.
- `nextflow.config`: archivo de configuración de parámetros globales y recursos.
- `run_deseq2.R`: script en R encargado del análisis de expresión diferencial.
- `run_goseq.R`: script en R encargado del análisis funcional de enriquecimiento.
